Плагин для перелинковки

== Description ==

Плагин позволяет вывести на сайт перелинковку, сгенерированную в сервисе perelink.pro

== Installation ==

1. Установите плагин через меню &#171;<strong>Плагины\Добавить новый</strong>&#187; (искать &#171;<strong>Perelink Pro</strong>&#187;).
2. Активируйте плагин в меню &#171;<strong>Плагины</strong>&#187;.
3. Зайдите в настройки плагина &#171;<strong>Настройки\Perelink Pro</strong>&#187;, введите защитный код из сервиса и сохраните настройки.
4. Это все.
